Your AI Safety Checklist

Print it. Pin it to your wall. Review it before adopting any new AI tool.

Quick reference Printable
Use This Daily

This checklist distills the entire guide into actionable items you can verify today. Print it, bookmark it, share it with your team.

Before You Use Any AI Tool

  • Is this tool on my organization's approved list?
  • If it handles sensitive data, is there a signed BAA?
  • Am I using the enterprise/API version, not the consumer version?
  • Do I understand what data the tool can access?
  • Have I enabled MFA on this account?
  • Am I using a unique, strong password from my password manager?

Your Devices

  • Is my disk encrypted? (FileVault / BitLocker — verify today)
  • Is my OS up to date?
  • Am I using a password manager or my browser's built-in password saving (with unique passwords per site)?
  • Do I lock my screen when stepping away?
  • Am I using a VPN on public WiFi?

Your AI Workflows

  • Am I pasting any PHI or sensitive data into AI tools? (If yes — is it BAA-covered?)
  • Have I audited my browser extensions for AI tools that read all pages?
  • Do I have any AI meeting bots or transcription tools running automatically?
  • Am I using any low-code automations (Zapier, Make) that touch sensitive data?
  • Have I reviewed what MCP servers are installed in my developer tools?

Quarterly Review

  • Disable AI tool accounts I'm no longer using
  • Review and update passwords
  • Check for new organizational policies on AI tool usage
  • Verify disk encryption is still active
Set a Reminder

Set a calendar reminder for the quarterly review. It takes 15 minutes and prevents most of the "I forgot I had that tool" incidents.

Quick Decision Tree

Want to use an AI tool with sensitive data?

→ Is it on the approved list?
NO → Stop. Get it approved first. Talk to IT or compliance.
YES → Does it have a signed BAA?
NO → Stop. Don't use with sensitive data. May still work for non-sensitive tasks.
YES → Are you using the enterprise/API version?
NO → Stop. Consumer ChatGPT, Claude, and Gemini are not BAA-covered.
YES → Is MFA enabled?
NO → Enable it now. Takes 2 minutes.
YES → Proceed with awareness. Review quarterly. Stay current on org policies.
You Made It

If you've read this far, you know more about AI safety than most people in your building. That's not a small thing. The tools are powerful, the risks are real, and the landscape changes fast — but the fundamentals don't. Approved tools, strong passwords, MFA, and a healthy skepticism about anything that asks for broad access to your data.

Keep this checklist somewhere you'll actually see it. Share it with a colleague. And if something comes up that this guide didn't cover, Ask Echs — the chatbot in the corner of every page — is there to help.
